## 모바일 반응형 디자인

> 현재 미디어 쿼리 전무. 사이드바가 고정 232px이라 모바일에서 레이아웃 전체 붕괴.
> 브레이크포인트 기준: `768px` (태블릿), `480px` (모바일)

### 1. 공통 레이아웃 ✅ — `App.module.scss` · `Sidebar` · `Topbar`

| 항목 | 작업 내용 |
|------|-----------|
| **앱 그리드** | 768px 이하 `grid-template-columns: 1fr` — 사이드바 그리드 컬럼 제거 |
| **사이드바 햄버거** | Topbar에 햄버거 버튼, 사이드바 `position: fixed` 드로어로 전환, 오버레이 추가 |
| **Topbar 패딩** | 768px 이하 `padding: 0 16px` |

### 2. 로그인 / 회원가입 / 비밀번호 찾기 ✅

| 항목 | 작업 내용 |
|------|-----------|
| **팀 선택 그리드** | 480px 이하 `repeat(2, 1fr)` |
| **카드 패딩** | 480px 이하 `padding: 28px 20px` |

### 3. 대시보드 (`/dashboard`) ✅ — `Dashboard.module.scss`

| 항목 | 작업 내용 |
|------|-----------|
| **12컬럼 그리드** | 768px 이하 모든 col을 `span 12` (full width 스택) |
| **식단 5열 그리드** | 768px 이하 가로 스크롤 (`minmax(88px, 1fr)`) |
| **heroStats** | `flex-wrap: wrap`, 구분선 숨김, stat 좌정렬 |

### 4. 캘린더 (`/calendar`) — `CalendarPage.module.scss` ← 난이도 높음

| 항목 | 현재 문제 | 작업 내용 |
|------|-----------|-----------|
| **7열 달력 그리드** | `repeat(7, 1fr)` 고정 — 모바일에서 셀 ~42px로 불가 | 768px 이하 주간/리스트 뷰 전환 또는 셀 높이 축소 + 칩 숨김 처리 |
| **셀 최소 높이** | `min-height: 112px` — 모바일에서 과도한 높이 | 480px 이하 `min-height: 56px` |
| **이벤트 칩** | 셀이 좁으면 칩 텍스트 잘림 | 480px 이하 칩 도트만 표시 |

### 5. 식단표 (`/meals`) — `MealsPage.module.scss`

| 항목 | 현재 문제 | 작업 내용 |
|------|-----------|-----------|
| **5열 그리드** | `repeat(5, 1fr)` — 모바일에서 각 카드 ~60px | 768px 이하 `repeat(3, 1fr)`, 480px 이하 `repeat(2, 1fr)` 또는 가로 스크롤 |
| **카드 최소 높이** | `min-height: 360px` — 모바일에서 뷰포트 초과 | 480px 이하 높이 제한 해제 |
